# Browser Semplice

Un browser web semplice e leggero che utilizza poca RAM, sviluppato in Python con Tkinter.

## Caratteristiche

- Interfaccia semplice con barra degli indirizzi
- Visualizza il contenuto delle pagine web come testo semplice
- Non carica immagini, CSS o JavaScript per ridurre l'uso di RAM
- Utilizza le librerie requests e BeautifulSoup per scaricare e parsare le pagine

## Requisiti

- Python 3.x
- Librerie: requests, beautifulsoup4

## Installazione

1. Installa le dipendenze:
   ```
   pip install -r requirements.txt
   ```

2. Esegui il browser:
   ```
   python main.py
   ```

## Uso

- Inserisci un URL nella barra degli indirizzi (es. https://www.google.com)
- Premi "Vai" per caricare la pagina
- Il contenuto verrà visualizzato come testo nell'area sottostante

Questo browser è ideale per navigazione leggera e lettura di contenuti testuali.
